# REST API for Movie Review Platform

Development of a REST API for a movie review platform where users can create movies, add them to favorites and create, edit, and delete reviews.

I was responsible for developing the backend as a REST API, while the frontend was implemented by a fellow student. A key learning outcome of this project was gaining practical experience in how the frontend and backend work together.

This project requires a MySQL database and environment variables.
Database credentials and API keys are **not included for security reasons**. 
Please configure your own environment to run the project.

## Technologies
- TypeScript
- Node.js
- Express
- MySQL

## Setup
- Copy .env.example to .env and fill in your own credentials and JWT secret.
- Import database.sql to create the required tables
